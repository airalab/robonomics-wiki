Para crear la interfaz de su dapp, puede elegir cualquier marco de JavaScript que le resulte cmodo o incluso intentar crear una interfaz sin ningn marco. En Robonomics 2021 usamos [Vue.js](https://vuejs.org) ya que es bastante escalable y fcil de usar. ## Configuracin para el Tutorial Si comienza con este paso y prefiere **aprender con la prctica**, siga esta lista de tareas para iniciar el dapp resultante de la leccin anterior: 1. Descargue un nodo Robonomics local de la [releases page](https://github.com/airalab/robonomics/releases/) que se adapte a su sistema operativo. Si no encuentra su sistema en la ltima versin, busque la versin ms reciente en las versiones anteriores. 2. Inicie el nodo Robononomics en el modo Desarrollador escribiendo `./robonomics --dev` en su terminal. 3. Descargue la extensin Polkadot para Chrome o Firefox [here](https://polkadot.js.org/extension/) 4. Clona [this repository](https://github.com/vol4tim/example-robonomics-dapp/). 5. Instale [Yarn](https://yarnpkg.com). 6. Instale [@vue/cli](https://cli.vuejs.org/guide/installation.html) 7. Comience a desarrollar dapp con el comando `yarn serve` en su terminal. **Debera tener esta pantalla en su navegador:** ![Dapp Start](../images/build-dapp-interface/dapp-start.png "Dapp Start")
Algunos consejos adicionales para el lanzamiento - Asegrate de que tu **nodo se est ejecutando**: ![Example of running a Robonomics node](../images/build-dapp-interface/robonomics-node-launch.png "Example of running Robonomics node") - En **macOS**, es posible que deba cambiar los **permisos de acceso** `chmod +x robonomics` - Asegrese de haber permitido **acceso para Polkadot Extension**: ![Polkadot Extension giving access](../images/build-dapp-interface/polkadot-permission.png "Polkadot Extension giving access") - Si tiene errores en el registro del nodo en ejecucin y dapp no se carga correctamente, intente eliminar la base de datos de la cadena de desarrollo: `sudo rm -rf /robonomics/chains/dev/db/` y reinicia el nodo. Si no ayuda, reinicie su mquina.

Hojas de Estilos Globales En este tutorial, muestro cmo cambiar la interfaz de una pequea dapp desde cero sin una biblioteca estable de componentes de UI. O si est de acuerdo con los temas de la interfaz estndar, puede usar cualquier biblioteca de interfaz de usuario de terceros ([por ejemplo](https://vuetifyjs.com/)). ## Primera Importacion o por donde empezar No tengo ningn diseo especfico para este dapp, pero tengo [Brandbook](https://static.robonomics.network/assets/Robonomics-Visual-Identity.pdf) y [tipografia](https://robonomics.network), las fuentes, los estilos de botones, etc. bien establecidos.
