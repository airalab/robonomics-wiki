    #Introduccin

    Este tutorial contina la leccin anterior, donde ya ha creado una aplicacin simple y se centr en conectar una cuenta a un nodo, enviar transacciones y otras funciones vitales de la dapp. Ahora crearemos una interfaz fcil de usar para esa aplicacin.

    #Prerrequisitos

    Este tutorial est diseado para personas que estn familiarizadas un poco con HTML, CSS, JavaScript y quieren aprender a aplicar estas habilidades para aplicaciones descentralizadas.

    Para crear la interfaz de su dapp, puede elegir cualquier marco de JavaScript que le resulte cmodo o incluso intentar crear una interfaz sin ningn marco. En Robonomics 2021 usamos Vue.js ya que es bastante escalable y fcil de usar.

    #Configuracin para el Tutorial

    Si comienza con este paso y prefiere aprender con la prctica, siga esta lista de tareas para iniciar el dapp resultante de la leccin anterior:

    1. Descargue un nodo Robonomics local v0.22 de la releases page que se adapte a su sistema operativo. Si no encuentra su sistema en la ltima versin, busque la versin ms reciente en las versiones anteriores.

    2. Inicie el nodo Robononomics en el modo Desarrollador escribiendo ./robonomics --dev --tmp en su terminal.

    3. Descargue la extensin Polkadot para Chrome o Firefox here

    5. Instale Yarn.

    6. Instale @vue/cli

    7. Comience a desarrollar dapp con el comando en su terminal:

    <template>
  <button type="button" :class="classList" @click="onClick" :disabled="disabled" class="inline-block">
    {{ label }}
  </button>
</template>

<script>

export default {

  components: {
    Icon: () => import("./Icon")
  },

  props: {
    label: {
      type: String,
    },
    type: {
      type: String,
      default: 'primary',
      validator: function (value) {
        return ['primary', 'secondary'].indexOf(value) !== -1;
      }
    },
    disabled: {
      type: Boolean,
      default: false,
    },
    size: {
      type: String,
      default: 'medium',
      validator: function (value) {
        return ['small', 'medium', 'large'].indexOf(value) !== -1;
      }
    }
  },

  computed: {
    classList() {
      return {
        'button': true,
        [`${this.type}`]: true,
        [`button__${this.size}`]: true,
      };
    },
  },

  methods: {
    onClick() {
      this.$emit('onClick');
    },
  },

};
</script>
